# 📊 Data Visualization Portfolio

A comprehensive data visualization portfolio demonstrating mastery of Python visualization libraries — **Matplotlib** and **Seaborn** — through 8 distinct, professionally styled chart types.

## 🎯 Project Overview

This project transforms raw data into compelling visual stories that enhance understanding and support decision-making. Each visualization section explores a different business scenario with curated color palettes and professional styling.

## 📈 Visualizations Included

| # | Visualization | Chart Type | Key Insight |
|---|--------------|-----------|-------------|
| 1 | Sales Performance | Grouped Bar Chart | Compare monthly sales across product categories |
| 2 | Revenue Trends | Multi-series Line Chart | Track quarterly revenue growth over 3 years |
| 3 | Market Share | Donut Chart | Visualize competitive market distribution |
| 4 | Student Performance | Correlation Heatmap | Discover hidden correlations in academic data |
| 5 | Salary Distribution | Violin + Box Plot | Analyze salary spread across departments |
| 6 | Ad Spend vs Revenue | Scatter + Regression | Identify spending-revenue relationships |
| 7 | Customer Satisfaction | Horizontal Stacked Bar | Survey response distribution analysis |
| 8 | Executive Dashboard | Multi-panel Dashboard | Combined KPI overview for leadership |

## 🛠️ Tools & Technologies

- **Python 3.x**
- **Matplotlib** — Core plotting library for bar charts, line charts, pie charts, dashboards
- **Seaborn** — Statistical visualization for heatmaps, violin plots, scatter plots
- **Pandas** — Data manipulation and analysis
- **NumPy** — Numerical computing and data generation
- **Jupyter Notebook** — Interactive development environment

## 🚀 Getting Started

```bash
# 1. Create virtual environment
python -m venv .venv

# 2. Activate virtual environment
.venv\Scripts\activate    # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Launch Jupyter Notebook
jupyter notebook Data_Visualization_Portfolio.ipynb
```
